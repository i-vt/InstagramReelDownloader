# InstagramReelDownloader

##  Build the image
`docker build -t instagram-scraper .`
## Run it
`docker run -p 5000:5000 --name igscraper instagram-scraper`

## Remove
Also to troubleshoot, you can remove the container and then re-run it `docker rm igscraper`
